# Notes

Point-in-time project artifacts — design briefs, audit outputs, progress logs, planning sessions.
**Working history, not canonical.** These captured a moment (a redesign, a ship night, an archive
audit); they were true when written and are kept for context, but they are *not* the source of
truth for how things work today.

> When something here conflicts with [`docs/`](../docs/README.md) (the editorial north star) or the
> per-app `CLAUDE.md` files (current runtime), **those win.** Verify against current code before
> trusting a claim in here.
