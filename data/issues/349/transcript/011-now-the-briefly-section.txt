There are eleven links this week.

Link one of eleven. "forge: A Python framework for self-hosted LLM tool-calling and multi-step agentic workflows"

Interesting project if you want to try running your own models. I haven't focused on self-hosted models yet. Better runtimes are what would make self-hosting actually accessible.

Link two of eleven. "tokenspeed — feel LLM tokens-per-second"

It is surprising to me how quickly I've gone from being amazed that Claude Code can do what it does to wishing it didn't take 15 minutes to do a complete refactor of a feature. This site does a good job of showing what token speed feels like.

Link three of eleven. "The Productivity Apps That Don’t Speak Agent Will Lose"

This is spot on. I'm looking at you OmniFocus! I also notice that apps are creating MCP bridges using AppleScript interfaces. Drafts recently did this. Unfortunately these are often not reliable and performant enough. I suspect the AppleScript in the middle is the problem. Agentic support needs to be direct.

Link four of eleven. "Introducing Shortcuts Playground: Create Apple Shortcuts with Claude Code or Codex"

This should be a built-in feature in Shortcuts right now.

Link five of eleven. "zerolang: The programming language for agents"

We've seen plenty of tools and software made for agents. How about a whole programming language designed for agents!

Link six of eleven. "The Emacsification of Software — Quarrelsome"

You can create as much bespoke software as you like these days. That is both amazing, and challenging at the same time. What a wonderful time to be a builder.

Link seven of eleven. "what 262,715 regex questions on stack overflow haven't answered | ian erik varatalu"

I'm a sucker for diving into rabbit holes filled with regular expressions.

Link eight of eleven. "How Claude Code works in large codebases: Best practices and where to start | Claude"

I'm using Claude in a monorepo (my first monorepo actually) and the multiple CLAUDE.md thing is an obvious but important thing to do.

Link nine of eleven. "A Field Guide to Learning - Bite of an Apple"

Five step approach to learn any topic.

Link ten of eleven. "feedr: A feature-rich terminal-based RSS/Atom feed reader written in Rust"

Seeing a terminal UI for RSS brought me back to the old days of use rn to read Usenet. If it supported syncing with Feedbin I might give it a try, but alas no.

Link eleven of eleven. "Review: ‘Steve Jobs in Exile’ recounts Apple founder’s tough mid-career lessons – Six Colors"

Learning the hard way.
