I realized it had been a while since I checked in on my [Dappnode](https://dappnode.com/en-us) that quietly chugs away running 32 validators for the [Gnosis](https://www.gnosischain.com) blockchain. It is monitored so I knew it was working but figured it was probably in need of some updates. Sure was!

<img src="https://www.thingelstad.com/uploads/2025/dappnode-start.png" alt="Dappnode Admin dashboard showing Gnosis chain sync status, machine stats at 13% CPU and 52°C, and four pending package updates including Core, Dms, Nethermind Xdai, and Teku Gnosis.">

I got a little shocked when I saw the uptime. 😬

<img src="https://www.thingelstad.com/uploads/2025/dappnode-uptime.png" alt="Dappnode uptime display showing up 1 year 28 weeks 3 days 12 hours 11 minutes">

This was much more behind than I thought it would be. I had some time so started the upgrades. I got some errors and had to reboot the Dappnode a couple of times to get updates to apply but eventually was able to get everything humming along nicely — never needing to do anything other than use the web browser.

<img src="https://www.thingelstad.com/uploads/2025/dappnode-done.png" alt="Dappnode Admin dashboard showing Gnosis chain nodes synced, CPU at 14% and 48C, memory at 46%, disk at 51%, all packages up to date.">

I'll try not to wait over a year to do the manual updates next.
