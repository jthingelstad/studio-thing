I've been using [Plausible Analytics](https://plausible.io/) for over a year and have been using the [Plausible Analytics Plug-in](https://github.com/LukasRos/plugin-plausible) created by [@lukas](https://micro.blog/lukas) to add it to my blog. It hasn't been updated for a long time and the embed code, while still working, is very outdated. I finally decided to take a few minutes to fork that plug-in and create my own [Plausible Analtyics 2](https://github.com/jthingelstad/microdotblog-plausible) for micro.blog.

It now uses the current code snippet and also sports a Plausible icon as well. Plus, I now have a plug-in in the directory. If you use Plausible Analytics on micro.blog I would definitely suggest switching.
