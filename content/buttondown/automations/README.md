# content/buttondown/automations/

Buttondown automation flow bodies and configs, version-controlled.

Each subdirectory is one automation. Suggested layout:

```
welcome-sequence/
├── automation.json          # Buttondown automation metadata
├── action-01-immediate.md   # body of the first action
└── action-02-day-7.md       # body of the second action
```

Sync scripts (`pull_automations.py`, `push_automations.py`) are not implemented yet.
