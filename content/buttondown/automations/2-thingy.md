subject: Meet Thingy, an AI agent for the archive
---

Weekly Thing has been running since 2017. That's a lot of issues to sift through if you're trying to remember when I wrote about something.

So I built **Thingy** — an AI agent that knows the archive. Ask it a question and it'll cite the specific issues it pulls from. "When did Jamie first write about RSS?" "What does he think about ActivityPub?" "Has Pinboard come up recently?" That kind of thing.

[Try Thingy](https://weekly.thingelstad.com/thingy/) — there are starter prompts on the page if you'd like a few ideas to begin with.

— Jamie
